# RESEARCH PROJECTS

## 1. On-Vehicle Generative AI Platform (Hyundai Mobis Grand Prize)
- **Goal** World-first in-car Edge generative AI infotainment  
- **Challenge** 24 TOPS NPU vs. Stable Diffusion requirements  
- **Solution**  
  - Visual CNN extracts road/weather/time “mood” features  
  - KD + pruning: model size −60 %, latency −45 %  
  - Multi-threaded feature-vector I/O; prompt templates for fast style shift  
- **Outcome** Grand Prize; pilot at Hyundai Mobis HQ

## 2. Multi-Sensory Soundscape App (GIST × Hyundai Motor, 2023 – present)
- Smartphone sensors (audio, IMU) → personalised functional soundscape  
- Feature-vector streaming; on-device DSP; 300 ms average latency

## 3. NE Robotaxi Soundscape AI (Hyundai Motor, 2022 – 2023)
- Real-time BGM adaptation for autonomous taxi; deployed on test fleet in US  
- FFT noise filtering; LAI model distillation; driver survey satisfaction 82 %

## 4. Office Soundscape AI (Hyundai Motor)
- Single AWS t4g instance handles 200 req/min; 80 % user satisfaction

## 5. GA-Driven Prompt Personalisation
- Surrogate model + GA improves FID −18 %, compute −40 % (TCICT 2024)

## 6. Diffusion Fine-Tuning for On-Device AI (Ph.D. dissertation)
- LoRA + KD pipeline; Adreno 740 GPU renders 512×512 image in 1.2 s
